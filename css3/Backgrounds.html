Backgrounds

Backgrounds 관련 속성

CSS3 에서는 엘리먼트의 배경을 좀 더 넓게 제어 할 수 있는 몇몇의 백그라운드 프로퍼티들을 가지고 있다.
하나의 엘리먼트에 여러개의 배경 이미지를 넣을 수 있는 속성과 함께 아래의 속성들도 함께 소개한다.

  • background-size
  • background-origin
  • background-clip

CSS3 Multiple Backgrounds

background-image 속성를 사용하면 하나의 엘리먼트에 여러개에 배경 이미지를 넣을 수 있다.
각각의 이미지를 콤마로 구분된다.

예제

background-image: url(img_flwr.gif), url(paper.gif);

background-position: right bottom, left top;

background-repeat: no-repeat, repeat;

CSS3 Background Size

background-size 는 배경 이미지의 크기를 지정하는 속성이다. CSS3 이전에는 배경 이미지의 크기는 실제 이미지의 크기 였다. CSS3 에서는 배경 이미지의 크기를 재지정 할 수 있다. 길이, 퍼센트로 지정이 가능하고 contain, cover 로 지정 할 수도 있다.

Full Size Background Image

브라우저 화면을 항상 배경 이미지로 덮고 싶을 때 사용한다.
이를 위해서는 아래 4가지를 만족해야한다

  • 페이지를 이미지로 채운다
  • 큰 이미가 필요
  • 이미지를 페이지 가운데 위치 시킴
  • 스크롤바가 생기지 않게 할 것
